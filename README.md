Landing page institucional para a **Gênesis Wood Work**, marcenaria artesanal que transforma madeira em peças únicas com propósito e fé.

Design dark & gold, minimalista e sofisticado — 100% responsivo, zero dependências.

## ✦ Features

| Feature | Descrição |
|---------|-----------|
| 🖼️ Hero Parallax | Imagem de fundo com efeito parallax suave no scroll |
| 🧱 Galeria Masonry | Layout adaptativo que respeita o tamanho real de cada foto |
| 📱 Mobile First | Navbar com hamburger menu + drawer lateral |
| 💬 WhatsApp Float | Botão flutuante de WhatsApp no mobile |
| ✨ Micro-animações | Hover com zoom, fade-in, bordas douradas na galeria |
| 🎨 Tipografia Premium | Playfair Display · Inter · Cormorant Garamond |
| ⚡ Performance | HTML + CSS + Vanilla JS — sem frameworks, sem dependências |

## 🚀 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/genesis-woodwork.git
```

2. Adicione suas imagens na pasta `img/`

3. Abra `index.html` no navegador — é só isso.

> Sem build, sem npm, sem configuração.
